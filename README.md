Projet BI Northwind
Ce projet est une application de Business Intelligence qui permet d'analyser les ventes et les livraisons de l'entreprise Northwind.
Il récupère automatiquement les données depuis SQL Server et des fichiers Excel, les nettoie, et les affiche dans un Dashboard interactif.
Fonctionnalités
Fusion de données : Combine les commandes récentes (SQL) et les archives (Excel).
Nettoyage automatique : Supprime les doublons et corrige les formats de date.
Tableau de bord : Visualisation des KPI (Commandes livrées, performance employés, Top clients).
Filtres : Analyse possible par période (date de début / fin).
Technologies utilisées
Python (Logique principale)
Pandas (Traitement des données)
SQL Server (Base de données)
Streamlit (Interface Web / Dashboard)
